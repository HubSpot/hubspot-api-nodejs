updateDraftTable

  • updateDraftTable(tableIdOrName, hubDbTableV3Request, includeForeignIds?, archived?, _options?): Promise<HubDbTableV3>

  • Update an existing HubDB table. You can use this endpoint to add or remove columns to the table as well as restore an archived table. Tables updated using the endpoint will only modify the draft verion of the table. Use publish endpoint to push all the changes to the published version. To restore a table, include the query parameter archived=true and \"archived\": false in the json body. Note: You need to include all the columns in the input when you are adding/removing/updating a column. If you do not include an already existing column in the request, it will be deleted. Update an existing table

      The ID or name of the table to update.

      The JSON schema for the table being updated.

    • Optional archived: boolean

    • Optional _options: Configuration
